# Functional Requirements

## Country Search

Users shall be able to search for countries.

## Health Dashboard

Users shall be able to view:

- Population
- Cases
- Deaths
- Recovered
- Active Cases

## Analytics

System shall calculate:

- Recovery Rate
- Mortality Rate
- Active Case Ratio
- Risk Score

## Comparison

Users shall compare multiple countries.

## Insights

System shall generate automated analytical summaries.

## Reporting

Users shall export reports.

---

# Non-Functional Requirements

## Performance

API responses should complete within 3 seconds.

## Reliability

Application should gracefully handle API failures.

## Maintainability

Codebase shall follow modular architecture.

## Security

No sensitive user data shall be collected.
